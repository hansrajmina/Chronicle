# **App Name**: Chronicle AI

## Core Features:

- AI-Assisted Writing: Expands user-written text using Google Gemini 2.5 Flash based on the current context, acting as a tool for content augmentation.
- Gamification: Earns XP and maintains streaks for consistent writing activity; data persists in Firestore.
- Humanizer: Rewrites text to improve the natural flow and tone.
- Language Support: Translates content into major Indian languages such as Hindi and Tamil.
- Reference Fetching: Automatically finds and cites academic and credible references to support the text, serving as an AI-powered research tool.
- Word Limit: Provides visual feedback to help writers meet their word count goals.
- PDF Export: Allows users to download their work as a PDF file for offline access and sharing.

## Style Guidelines:

- Primary color: Electric blue (#7DF9FF) for a stark, high-contrast aesthetic.
- Background color: Soft gray (#F0F0F0) to provide a neutral base for content.
- Accent color: Darker gray (#A9A9A9) to highlight interactive elements without overwhelming the user.
- Body and headline font: 'Inter', a sans-serif font that supports the minimalist, Cred-like aesthetic, working for both headlines and body.
- Card-based widgets with soft but deep shadows for a Neobrutalist interface.
- Use bold, uppercase labels with high contrast.
